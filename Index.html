__चिड़िया उड़__

Relive the nostalgic joy of childhood!

Hey there! The game 'Chidiya Ud' sounds like a fun childhood memory! In this game, players take turns naming differentthings like birds, cars, or stationary items. The catch is that after naming an item, the player must add 'ud' to the endof it if it can fly. If a player forgets to add 'ud' to a flying item or mistakenly adds it to a non-flying item, theylose that round. It's a simple yet engaging game that tests memory and quick thinking. Enjoy reminiscing about thoseplayful moments!
